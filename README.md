# Moid Addon 🌑

A specialized utility expansion for the **Meteor Client** (1.21.1), designed with a focus on performance, internationalization, and "vibe-coded" logic.

---

## 🛠️ Installation

1. **Prerequisites:** Ensure you have [Meteor Client](https://meteorclient.com/) installed for Fabric 1.21+.
2. **Download:** Go to the [Releases](https://github.com/DioM34/Moid-Addon/releases) tab and download the latest `.jar` file.
3. **Move:** Drop the `.jar` into your Minecraft `mods` folder.
4. **Launch:** Start Minecraft and enable **Moid Addon** in the Meteor modules menu.

---

## 🧩 Featured Modules

| Module | Description |
| :--- | :--- |
| **Moid-Backtrack** | Freezes entity positions by delaying inbound movement packets. Includes a **Ghost Renderer** to show the server-side position for easy hits. |
| **Moid-HUD** | A highly customizable, performance-friendly HUD. Shows FPS, Ping, Coords, and a stylish active-module list. |
| **Moid-PingSpoof** | Simulates lag to the server. Features **Static** and **Randomize** modes to create "jitter" and mask your actual latency. |
| **Moid-HitESP** | Visualizes where your hits are landing with customizable color-fade boxes. |
| **Moid-TriggerBot** | Automatically attacks entities when they enter your crosshair with precision timing. |
| **Moid-Velocity** | Prevents or modifies knockback received from players and the environment. |
